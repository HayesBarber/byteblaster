## Installation

**Prerequisites**: Zig 0.16

Clone the repo

```bash
git clone https://github.com/HayesBarber/byteblaster.git
cd byteblaster
```

Run with zig

```bash
zig build run
```

or install to a specified location

```bash
zig build -p /some/path
```
